# Démarrage rapide

Cette page est en cours de rédaction. La documentation complète sera bientôt disponible.

## Installation

Ajoutez le plugin HexaGlue à votre `pom.xml` :

```xml
<plugin>
    <groupId>io.hexaglue</groupId>
    <artifactId>hexaglue-maven-plugin</artifactId>
    <version>0.1.0</version>
    <executions>
        <execution>
            <goals>
                <goal>generate</goal>
            </goals>
        </execution>
    </executions>
</plugin>
```
